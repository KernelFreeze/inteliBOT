Welcome to inteliBOT 1.0 Cream.

This automatic tool for edit wiki sites. To configure, please use the follow commands:

$ cd inteliBOT
$ git pull
$ .configure
$ make
$ cd ./inteliBOT/bin/debug
$ mono inteliBOT.exe

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch and compile with make.
